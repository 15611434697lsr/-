# 交通计数器 v1.0.0 正式版

交通计数器是一款面向交通调查、道路工程、课程实践和车流量数据整理的轻量化工具。网站版支持在线人工计数，本地 Windows 软件提供更完整的辅助识别和数据处理能力。

## 在线使用

- 网站地址：[https://15611434697lsr.github.io/-/](https://15611434697lsr.github.io/-/)
- 网页版计数器：[https://15611434697lsr.github.io/-/counter.html](https://15611434697lsr.github.io/-/counter.html)
- Windows 软件包：[下载 trafficcounter.zip](https://github.com/15611434697lsr/-/releases/latest/download/trafficcounter.zip)

## 主要功能

- 断面交通量人工计数
- 交叉口转向交通量人工计数
- 进口道方向与车道配置
- 本地软件支持视频识别辅助分析
- 本地软件支持车速估计、Excel 导出和交通量换算
- 适用于交通调查、课堂实训、课程设计、道路工程资料整理等场景

## 网页版与本地版区别

网页版适合快速人工计数，打开网页即可使用，不需要安装。为了保持网页端轻量，网页版不提供 Excel 导出、标准交通量换算、视频识别和车速估计。

本地 Windows 软件适合需要完整功能的场景。下载压缩包后，请完整解压文件夹，再运行其中的 exe 程序。

## 使用说明

1. 打开网站首页。
2. 点击“在线使用网页版计数器”进入人工计数页面。
3. 如需视频识别、Excel 导出或完整数据处理，点击“下载 Windows 软件”。
4. 解压下载的 `trafficcounter.zip`，运行文件夹中的 exe 程序。
